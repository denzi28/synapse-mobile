# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

**Synapse mobile app:** see [`synapse-mobile/README.md`](synapse-mobile/README.md) for Expo setup, EAS builds, and current behavior (English default, Shots, onboarding, etc.).

View your app in AI Studio: https://ai.studio/apps/cd17b3da-76f6-4f98-b0ab-f7e6147f8962

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
